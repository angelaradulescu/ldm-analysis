Look at Learning over Age

Next, we run a simple linear model to see if there is an effect of age on the number of games that a subject has learned. We compute number of games learned as the number of games that a subjects point of learning (100% accuracy in all subsequent trials) is before the 15th trial. We regress the sum stored in GamesLearned on continuous age and see no significant correlation.
